Will initialize the kingdom and set the following subject as king

            
const KING = {
    element: 'div',
    textAsHTML: ' I am the KING! ',
    setAttributes:{
        'class': 'king lord item'
    }
}
kingdom = new Kingdom({...KING});

Adds a subject to the current lord

                
const SUBJECT = {
    element: 'div',
    textAsHTML: ' I am your loyal subject! ',
    setAttributes:{
        'class': 'subject item'
    }
}
kingdom = kingdom.addSubject({...SUBJECT});

Creates new subject. Sets them as current lord

                
const SUBJECT_LORD = {
    element: 'div',
    textAsHTML: ' I am your loyal subject, but I too rule! ',
    setAttributes:{
        'class': 'subject lord item'
    }
}
kingdom = kingdom.addSubject({...SUBJECT_LORD}, true);
